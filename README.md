## 字符串大小写转换
```python
str.upper()        # 把所有字符中的小写字母转换成大写字母
str.lower()        # 把所有字符中的大写字母转换成小写字母
str.capitalize()   # 把第一个字母转化为大写字母，其余小写
str.title()        # 把每个单词的第一个字母转化为大写，其余小写 
```

## 格式符
```python
%s    #字符串 (采用str()的显示)

%r    #字符串 (采用repr()的显示)

%c    #单个字符

%b    #二进制整数

%d    #十进制整数

%i    #十进制整数

%o    #八进制整数

%x    #十六进制整数

%e    #指数 (基底写为e)

%E    #指数 (基底写为E)

%f    #浮点数

%F    #浮点数，与上相同

%g    #指数(e)或浮点数 (根据显示长度)

%G    #指数(E)或浮点数 (根据显示长度)
```
